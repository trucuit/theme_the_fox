# Theme The Fox

WordPress theme tùy chỉnh — giao diện đẹp, responsive, phù hợp cho blog cá nhân.

## 📋 Tính năng

- Responsive design
- Custom WordPress theme
- Hỗ trợ blog layout
- Cross-browser compatible

## 🛠️ Công nghệ

- **CMS:** WordPress
- **Language:** PHP, HTML, CSS, JavaScript

## ⚙️ Cài đặt

1. Copy thư mục theme vào `/wp-content/themes/`
2. Kích hoạt theme trong WordPress Admin → Appearance → Themes
