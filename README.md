# Theme The Fox

Bài tập cắt giao diện web HTML/CSS/JS từ design template "The Fox".

## 🌐 Demo

**Live:** [https://trucuit.github.io/theme_the_fox/](https://trucuit.github.io/theme_the_fox/)

## 📋 Tính năng

- Responsive design (Bootstrap)
- Carousel banner
- Blog cards layout
- Featured posts section
- Social media links
- Font Lato custom

## 🛠️ Công nghệ

- **Frontend:** HTML, CSS, JavaScript
- **Framework:** Bootstrap 4
- **Font:** Lato (Google Fonts)
- **Icons:** Font Awesome

## 📁 Cấu trúc

```
theme_the_fox/
├── index.html     # Trang chính (static)
├── 1.css          # Stylesheet
├── 1.js           # JavaScript
├── Lato/          # Font Lato
├── fonts/         # Font files
├── images/        # Hình ảnh
├── vendor/        # Bootstrap, jQuery, Font Awesome
└── file/          # HTML partials
```

## 🔒 Bảo mật

Static site, không yêu cầu API key hay credentials.
